## `JAIROCloudGroupsManager`
当機能のFlask拡張を提供するクラス。

#### 属性

| 属性名 | 型                                                 | 説明                                           |
| ------ | -------------------------------------------------- | ---------------------------------------------- |
| app    | Flask \| None                                      | Flaskアプリケーションインスタンス              |
| config | str \| [RuntimeConfig](02.config.md#runtimeconfig) | サーバー設定値のインポート名またはオブジェクト |

#### メソッド

| 名前                     | 説明                                      |
| ------------------------ | ----------------------------------------- |
| &#95;&#95;init&#95;&#95;() | クラスの初期化メソッド                    |
| init_app()                 | Flaskアプリケーションを初期化するメソッド |
| init_config()              | サーバー設定値を初期化するメソッド        |
| init_db_app()              | データベースを初期化するメソッド          |


### `__init__(self, app: Flask, config: str | RuntimeConfig) -> None`
クラスの初期化メソッド。

#### 引数

| 名前   | 型                                                       | 説明                                           |
| ------ | -------------------------------------------------------- | ---------------------------------------------- |
| app    | Flask                                                    | Flaskアプリケーションインスタンス              |
| config | str \| [RuntimeConfig](02.config.md#runtimeconfig) | サーバー設定値のインポート名またはオブジェクト |

#### 処理内容
1. 属性 `app` に引数 `app` を設定する。
2. 属性 `config` に引数 `config` を設定する。引数の指定がなければ、[DEFAULT_CONFIG_PATH](./03.const.md#default_config_path-str) を設定する。
3. 引数 `app` が指定されている場合、[init_app](#init_appself-app-flask-none) メソッドを呼び出す。


### `init_app(self, app: Flask) -> None`
Flaskアプリケーションを初期化するメソッド。

#### 引数

| 名前 | 型    | 説明                              |
| ---- | ----- | --------------------------------- |
| app  | Flask | Flaskアプリケーションインスタンス |

#### 処理内容
1. 属性 `app` に引数 `app` を設定する。
2. [init_config](#init_configself-app-flask-none) メソッドを呼び出し、サーバー設定値を初期化する。
3. [init_db_app](#init_db_appself-app-flask-none) メソッドを呼び出し、データベースを初期化する。
4. ロガーを設定する。


### `init_config(self, app: Flask) -> None`
サーバー設定値を初期化するメソッド。

#### 引数

| 名前 | 型    | 説明                              |
| ---- | ----- | --------------------------------- |
| app  | Flask | Flaskアプリケーションインスタンス |

#### 処理内容
1. 属性 `config` の値で [setup_config](02.config.md#setup_configpath_or_obj-str-runtimeconfig-runtimeconfig) 関数を呼び出し、
  サーバー設定値を初期化する。
1. 引数 `app` の設定オブジェクトにサーバー設定値を登録する。
2. 引数 `app` の設定オブジェクトに環境変数を読み込ませる。


### `init_db_app(self, app: Flask) -> None`
データベースを初期化するメソッド。

#### 引数

| 名前 | 型    | 説明                              |
| ---- | ----- | --------------------------------- |
| app  | Flask | Flaskアプリケーションインスタンス |

#### 処理内容
1. データベース拡張機能を初期化する。
2. モデルをロードする。
